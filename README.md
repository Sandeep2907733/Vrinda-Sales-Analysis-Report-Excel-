# Vrinda Sales Analysis Report (Excel)

# Overview
This project analyzes Vrinda Store sales data using Microsoft Excel to uncover trends, customer behavior, and key business insights. The report converts raw data into a structured and interactive Excel dashboard for decision-making.

# Objectives

Analyze overall sales performance

Identify category-wise sales trends

Understand customer demographics

Track monthly sales patterns

# Tools Used

Microsoft Excel

Data Cleaning

Pivot Tables & Pivot Charts

Slicers & Filters

Dashboard Design

# Key Insights

Monthly and category-wise sales analysis

Customer analysis by age and gender

Channel-wise and regional performance

# Features

Interactive Excel dashboard

Dynamic slicers for filtering

Clear and business-focused visuals

# Learnings

Practical experience in Excel-based data analysis

Improved data visualization and storytelling

Hands-on dashboard creation
